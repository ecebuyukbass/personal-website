## Personal Website Platform

This project demonstrates backend architecture design, authentication logic, database integration, and frontend integration in a modular full-stack setup.

## Features

Role-based Authorization (Admin Guard),
Article Management Module,
RESTful API structure,
Prisma ORM with PostgreSQL,
Environment-based configuration,
Frontend & Backend separation.

## Tech Stack
Backend :
NestJS,
Prisma ORM,
PostgreSQL,
Modular architecture

Frontend:
React,
Vite,
Modular CSS.
